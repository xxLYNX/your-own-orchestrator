🚀 QUICK START COMMANDS:

  Add your first note:
    ./bin/yoo add "My first task"

  Add a note with priority:
    ./bin/yoo add "Important meeting" --priority high

  Add a note for a specific date:
    ./bin/yoo add "Dentist appointment" --date 2024-06-15

  Add a note with tags:
    ./bin/yoo add "Review PR" --priority high --tags work,urgent

  View today's schedule (launches TUI):
    ./bin/yoo schedule

  View a specific date:
    ./bin/yoo schedule --date 2024-06-15

  Install globally:
    make install

  After installing, use anywhere:
    yoo add "task"
    yoo schedule
