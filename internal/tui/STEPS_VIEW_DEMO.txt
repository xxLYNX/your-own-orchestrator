--------------------------------------------------------------------------------
                              COLOR REFERENCE
--------------------------------------------------------------------------------

The actual TUI uses the following color scheme:

Primary Color:   #7D56F4 (Purple) - Headers, highlights, cursor
Success Color:   #04B575 (Green)  - Completed items, checkmarks, progress
Warning Color:   #F59E0B (Orange) - Pending status, notes
Muted Color:     #666666 (Gray)   - Help text, timestamps
Subtle Color:    #999999 (L-Gray) - Descriptions, secondary text
Border Color:    #444444 (D-Gray) - Borders and dividers

Icons Used:
✓ ☑ - Completed checkbox
○ ☐ - Unchecked checkbox
> - Cursor/selection indicator
→ - Arrow/output indicator
📋 - Template/checklist icon
🕐 - Time/duration icon
📅 - Date icon
⏳ - Pending status
▶ - In progress status


--------------------------------------------------------------------------------
                          INTERACTION EXAMPLES
--------------------------------------------------------------------------------

1. TOGGLE STEP COMPLETION
   - Position cursor on step using ↑/↓ or j/k
   - Press Space or Enter
   - Step immediately marks as complete/incomplete
   - Progress bar updates in real-time
   - Completion timestamp added/removed

2. ADD NOTE TO STEP
   - Position cursor on desired step
   - Press 'n' to enter note mode
   - Type your note (existing note pre-populated)
   - Press Enter to save, Esc to cancel
   - Note appears in step details panel

3. VIEW DETAILED INFORMATION
   - Position cursor on any step
   - Press 'v' to toggle detailed view
   - See full description, checklist, times, notes
   - Press 'v' again or Esc to return to list

4. NAVIGATE STEPS
   - Use arrow keys (↑/↓) or vim keys (j/k)
   - Cursor wraps at top/bottom
   - Selected step details update automatically
   - Strikethrough applied to completed steps

5. EXIT VIEW
   - Press 'q' or Esc to quit
   - All changes are auto-saved to database
   - Returns to previous view/terminal
